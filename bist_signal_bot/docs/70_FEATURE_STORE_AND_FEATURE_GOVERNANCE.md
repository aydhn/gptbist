# Local Feature Store and Feature Governance

The local feature store manages all signals and research metrics deterministically and safely.

## Architecture
- **Contracts**: Schemas preventing regressions.
- **Sets**: Grouped features with versioning.
- **Frames**: Point-in-time cross-sectional matrices.
- **Serving**: ML and Scanner delivery.
- **Leakage Guard**: Prevents looking ahead.
- **Drift**: Monitors statistical divergence.

## Important Note
This is a research-only environment. Outputs are not trade recommendations.
